== Description ==
MyBlog is a super modern, animated WordPress theme built with React, TailwindCSS, and Framer Motion. 
It features a responsive design, smooth animations, and a clean, minimalist aesthetic.

== Installation ==
1. In your WordPress admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the myblog.zip file.
3. Click Install Now.
4. Click Activate to use your new theme right away.

== Features ==
* React-powered frontend
* TailwindCSS for styling
* Framer Motion animations
* Fully responsive design
* Dark mode support
* Blog functionality with WordPress REST API
* Contact form integration
* SEO friendly

== Configuration ==
After activation:
1. Go to Settings > Permalinks and select "Post name" for pretty URLs
2. Create pages for Home, Blog, About, and Contact
3. Set your homepage in Settings > Reading
4. Configure menus in Appearance > Menus
